集合与映射

  • 集合论是德国著名数学家康托尔于19世纪末创立的
  • 牛顿与莱布尼兹在十七世纪创立了微积分并获得飞速发展
  • 十八世纪，由于无穷概念没有精确的定义，使微积分理论遇到眼中的逻辑困难
  • 十九世纪初，出现了一场重建数学基础的运动，集合论由此产生

集合的概念与运算

1.集合的定义

将具有某种特定性质的对象的全体称为集合。组成集合的对象称为元素
a ∈ A ， a ∉ B

  • 集合的表示法
    1. 枚举法：A = {a,b,c,d,e}

      A = {3,5,7,9,11}

    2. 描述法: A = {x|x所具有的特征}

      A = {x|x为前5个素数}

  • 集合的关系
    • 子集：A⊂B
    • 相等：A = B
    • 空集：Φ
  • 数集的表示
    • 自然数 N = { x | x = 1, 2, …}
    • 整数 Z = { x | x = 0, ±1, ±2,…}
    • 有理数 Q = {p/q | p,q ∈ Z,q ≠ 0}
    • 实数 R
    • 复数 C
    • 复平面 z = x + iy ∈ C
集合运算

并集 A ∪ B = { x | x ∈ A 或 x ∈ B}
交集 A ∩ B = { x | x ∈ A 且 x ∈ B}
差集 A - B = { x | x ∈ A 且 x ∉ B}

确界与连续性公理

  • 实数的性质
    • 有序性
    • 连续性或完备性
  • 数集的上界与上确界
    • 设E是一个非空实数集，M是一个实常数，如果对于E中的人和元素x，均有 x ≤ M，则称 M 为数集E的一个上界，并称E有上界。
    • 如果一个实数集E有上界，称E的最小上界为上确界，记为sup E (supremum的缩写)
    • 连续性公理：一个非空有上界的实数集必有上确界。
    • 设E是一个非空实数集，m是一个实常数，如果对于E中的任何元素x，均有 x ≥ m,则称 m 为数集E的一个下界，并称E有下界。
    • 如果一个实数集E有下界，称E的最大下界为下确界，记为inf E(infimum的缩写)
    • 连续性公理：一个非空有下界的实数集必有下确界。

微积分的基础是极限理论，而连续性公理是极限理论的基石。

例2 讨论有限集与无限集的上下界和上下确界，并判断上下确界是否在集合内。
集合A有上界也有下界
上界：7以及比7更大的数 supA = 7
下界: 2以及比2更小的数 infA = 2
集合B有上界也有下界
上界：1以及比1更大的数 supB = 1
下界: 小于或等于零的数 infB = 0

区域与邻域

  1. 区间
    • 有限区间
      开区间 (a,b) = { x | a < x < b }
      闭区间 [a,b] = { x | a ≤ x ≤ b }
      半开闭区间 [a,b) = { x | a ≤ x ≤ b } 和 (a,b] = { x | a < x ≤ b}
    • 无限区间
      无限开区间 ( -∞，a ) = { x | x < a } 和 ( a, +∞ ) = { x | x > a}
      无限闭区间 ( -∞,a ] = { x | x ≤ a } 和 [a,+∞) = { x | x ≥ a}
      全体实数的集合 R = (-∞，+∞)
  2. 邻域
    邻域: 以点a为中心的人和开区间，记作: U(a)
    δ 邻域: U(a,δ) = (a-δ,a+δ) = { x | |x-a| ＜ δ}
    去心邻域: U(a,δ)
