Solr是基于Lucene的全文搜索服务器

安装和配置

Solr配置到tomcat

  1. 把Solr的war包复制到tomcat下的webapp下解压
  2. 把Solr/example/lib/ext目录下的jar包添加到solr工程中
  3. 配置SolrHome和SolrCore
    • solr/example/solr 文件就是一个标准的solrhome, 复制重命名solrHome
    • solrHome中的collection1就是一个solrCore
    • solrCore下有一个目录conf，conf下的solrconfig.xml可以配置相关信息
  4. solrconfig.xml
    • env-entry-value:配置solrhome的绝对路径
    • lib:solr服务依赖的拓展包
    • dateDir:配置索引库存放路径
    • requestHandler:查询时使用的url

  5. 打开Schema.xml可以看到solr默认的FieldType

    • class:solr提供的包solr.TextField。solr.TextField允许用户通过分析器来定制索引和查询，分析器包括一个分词器(tokenizer)和多个过滤器(filter)
    • positionlncrementGap:可选属性，定义在同一个文档中此类型数据的空白间隔，避免短语匹配错误，此值相当于Lucence的短语查询设置slop值，根据经验设置为100
    • analyzer
      • 搜索分析器
        • solr.StandardTokenizerFactory标准分词器
        • solr.StopFilterFactory停用词过滤器
        • solr.LowerCaseFilterFactory小写过滤器
      • 索引分析器
        • solr.StandardTokenizerFactory标准分词器
        • solr.StopFilterFactory 停用词过滤器
        • solr.SynonymFilterFactory同义词过滤器
    • Field定义
      • name 域名
      • type FieldType
      • indexed 是否索引
      • stored 是否存储
      • multiValued 是否存储多个值
    • uniqueKey
      默认定义唯一主键key为id域
      <uniqueKey>id</uniqueKey>

    • copyField复制域
      将多个Field复制到一个Field中，进行统一检索
      比如，输入关键字搜索name/description

    • dynamicField(动态字段)
      自定义Field为:product_title_t
      <dynamicField name="*_t" type="text_ik" indexed="true" stored="true"/>

配置中文分析器

  1. 将IKAnalyzer2012FF_ul.jar添加到solr/WEB-INF/lib
  2. 复制IKAnalyzer的配置文件和自定义词典到solr的classpath下

  3. 在schema.xml中添加一个自定义的fieldType

  4. 定义filed，指定field的type属性为text_ik

查询索引

  • q- 查询字符串 域名:条件
  • fq- filter query 过滤查询
  • sort- 排序
  • start,rows- 分页查询
  • fl- 返回指定字段内容，用逗号或空格分割
  • df- default Feild 默认域
  • wt- writer type 指定输出格式
  • hl- 是否高亮
